## Image Guidelines

When adding images to this folder:
- Use descriptive filenames
- Prefer PNG for diagrams and screenshots
- Use JPG for photos
- Keep file sizes reasonable (< 2MB)
- Add captions in README files where images are used

## Contributing Documentation

To improve documentation:
1. Add photos of your build
2. Create wiring diagrams
3. Add troubleshooting guides with images
4. Submit pull requests with your additions
